# sim

Simple vIM. Why? Why not?

Text Editor with vim keys made with Ncurses. ~~Currently only works as a pager~~ ;)

### Keys Supported 

```
navigation: h, j, k, l, gg, G

```
 
### Dependencies

+ [Ncurses](https://invisible-island.net/ncurses/#download_ncurses)

### Build

`./build.sh`

### Run

`./bin/out main.cpp`
